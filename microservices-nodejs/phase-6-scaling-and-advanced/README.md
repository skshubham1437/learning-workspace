# Phase 6: Scaling & Advanced Patterns (~2 weeks)

## Modules

### Module 12 — Scaling Patterns
Horizontal scaling. Load balancing algorithms. Caching at every layer. Distributed caching. Read replicas. Database sharding. Auto-scaling triggers and thresholds.

### Module 13 — Advanced Architectural Patterns
Event-Driven Architecture. Hexagonal Architecture. Clean Architecture in microservices. Strangler Fig Pattern. Sidecar Pattern. Ambassador Pattern. Anti-Corruption Layer.

## Goal
By the end of this phase, you should be able to scale individual services under load and apply advanced patterns to evolve your architecture over time.
