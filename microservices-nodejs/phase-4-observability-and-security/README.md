# Phase 4: Observability & Security (~1.5 weeks)

## Modules

### Module 08 — Observability
Structured logging across services. Metrics (Prometheus). Distributed tracing (OpenTelemetry, Jaeger). Full observability stack. Correlating logs, metrics, and traces during incidents.

### Module 09 — Security
Service-to-service auth. mTLS. JWT in microservices (vs monolith usage). Secrets management (Vault, K8s secrets). API security at gateway level. Zero-trust networking.

## Goal
By the end of this phase, you should be able to answer "what's happening across 20 services right now?" and "who's allowed to talk to whom?" with confidence.
