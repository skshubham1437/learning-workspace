# Phase 7: Capstone & System Design (~3 weeks)

## Modules

### Module 14 — Production Case Study
Build a complete e-commerce platform microservices system from scratch, starting from a monolith. Services: User, Product, Inventory, Cart, Order, Payment, Notification. Stack: Node.js, PostgreSQL, Redis, RabbitMQ/Kafka, Docker, Kubernetes, OpenTelemetry.

### Module 15 — System Design Thinking
CAP theorem applied to real decisions. PACELC. Consistency models (strong, eventual, causal). Common distributed system failure modes. Structuring Staff-level system design interview answers.

## Goal
By the end of this phase, you should have a portfolio-worthy microservices project and the ability to walk through any system design interview with structured, tradeoff-aware reasoning.
