# Phase 3: Edge & Reliability (~1.5 weeks)

## Modules

### Module 06 — API Gateway & Edge Patterns
API Gateway design. Backend For Frontend (BFF). Aggregator Pattern. Service Mesh basics. Comparing Kong, NGINX, Traefik, Envoy. Service Discovery: static vs dynamic, client-side vs server-side discovery, Consul, AWS Cloud Map, Kubernetes kube-dns.

### Module 07 — Reliability Patterns
Circuit Breaker. Retry with exponential backoff. Timeout strategies. Bulkhead isolation. Rate limiting. Dead Letter Queues. Backpressure. Cascading failure recovery. Graceful degradation strategies. Fallback responses and feature flags under failure.

## Goal
By the end of this phase, you should be able to build systems that degrade gracefully instead of catastrophically — the difference between a 3am page and a minor blip.
