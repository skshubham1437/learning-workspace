# Learner struggles with problem solving and wants to tackle LeetCode

The user explicitly stated: "i'm not good at problem solving skills and i also wants to solve leetcode questions". 

This is the most common and important hurdle in DSA. Jumping straight into learning "Trees" or "Graphs" will cause frustration because the core issue is not a lack of data structures knowledge, but a lack of a systematic approach to breaking down a problem.

## Implications
- **Lesson 1 must not contain any code.** It must focus entirely on a problem-solving framework (UMPIRE) to give the user a tool to use when their mind goes blank.
- Future lessons must enforce the "Plan" stage before showing the "Implement" stage.
- Emphasize pattern recognition over volume of problems solved.
